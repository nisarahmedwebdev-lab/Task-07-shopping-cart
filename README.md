# 🛒 TechStore - PHP Session Based Shopping Cart

A modern mini e-commerce shopping cart system built with **PHP, HTML, CSS, and Vanilla JavaScript**.  
The cart is fully session-based and works without authentication, database, or external libraries.

## 🚀 Features

- Responsive modern e-commerce UI
- Product listing with dynamic PHP rendering
- PHP Session based shopping cart
- Add products to cart using AJAX
- Update product quantity (+ / -)
- Remove products from cart
- Live cart badge counter
- Flyout shopping cart sidebar
- Smooth hero carousel slider
- Automatic slide movement
- Manual next/previous slider controls
- Real-time cart updates without page reload
- Checkout button interaction
- Mobile, tablet, and desktop responsive design
- Modern footer section

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- PHP
- PHP Sessions
- AJAX
- Font Awesome Icons
